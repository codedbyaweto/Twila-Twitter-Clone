# Twimba - A Twitter Clone

Twimba is a lightweight Twitter-style microblogging app built with HTML, CSS, and JavaScript (ES Modules).  
It allows users to post tweets, like and retweet posts, and view threaded replies - all rendered dynamically in the browser.

---

## Features
- Create and post new tweets in real time  
- Like and unlike tweets  
- Retweet functionality with live counter updates  
- Toggle replies for individual tweets  
- Dynamic feed rendering using JavaScript  
- Modular ES6 code structure  
- Unique tweet identifiers generated with `uuid`

---

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6 Modules)  
- Font Awesome (for icons)  
- Google Fonts (Roboto)  
- UUID (for unique ID generation)
